## 初見
 - 把字串作為array, swap頭尾兩元素直到陣列中央即完成反轉
```javascript=
var reverseString = function(s) {
    let result = s;
    for(let i=0; i<s.length/2; i++){
        [s[i], s[s.length-i-1]] = [s[s.length-i-1], s[i]]
    }
    return result;
};
```
## 備註
- javascript 字元編碼格式為UTF16
